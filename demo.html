how to send

// before sending · read

  • Styles we sign: dark progressive, minimal techno com pitada de psy, neuro, minimal psytrance, forest, experimental techno. We don't reply outside this spectrum.
  • Demo: send 2-4 tracks max · WAV / FLAC / MP3 320kbps · streaming via SoundCloud private link (preferred) or Dropbox / WeTransfer.
  • Don't publish the tracks as a release before our reply. Tracks already released on other labels are not accepted.
  • Bio + socials: include a short paragraph about the project and SoundCloud / Bandcamp / Instagram links.
  • Reply: average 3-6 weeks. A&R: Tekmall, Anaïs Lin, Mallki, Nuit Noire and rotating curators.
